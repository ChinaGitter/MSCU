微软商学院技术精英研修班

​​2015 年伊始，微软商学院特举办技术精英研修班系列活动，每月将有针对不同产品的技术培训。

2015-1​

2015 年伊始，微软商学院特举办技术精英研修班系列活动，每月将有针对不同产品的技术培训。
二月初，微软商学院成功举办了Windows Azure 动手实验营，与学员携手步入 Windows Azure 云世界，了解国际最新资讯，全程中文深入讲解。来自微软全球合作伙伴服务部技术顾问带学员深入了解 Windows Azure 的功能及细节，内容包括如何在新时代 IDC中快速、弹性、经济地实现网络、身份验证、网站、数据库等服务。更有讲师指导学员动手操作中文实验环境，真切体会云技术，了解云价值。
目标受众
企业架构师，技术工程师或开发人员，具备 IaaS (云基础架构即服务） 和 PaaS（平台即服务）的项目经验，无年限要求。同时具备：
- 构建网络，Hyper-V，高可用，数据中心项目的经验
- 理解 Web Service 概念，最好具有相关开发经验
- 基础的 SQL Server 和 SharePoint 的知识
培训收益
- 了解了最新的 Windows Azure 功能和特性，传统数据中心的服务如何迁移 到 Windows Azure 云数据中心，各种服务与 Windows Azure 云服务的 匹配关系。
- 深入理解了新时代的 IDC，如何在云端实现网络、身份验证、网站、数据库等服务。
